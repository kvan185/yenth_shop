WEB CỜ TƯỚNG 2 NGƯỜI

Cách chạy:
1. Giải nén file ZIP.
2. Mở file index.html bằng Chrome, Edge hoặc Firefox.
3. Hai người chơi cùng một máy, lần lượt chọn quân rồi chọn điểm đến.

Tính năng:
- Bàn cờ tướng 9x10.
- Quân đỏ và quân đen.
- Đi đúng luật cơ bản: Tướng, Sĩ, Tượng, Mã, Xe, Pháo, Tốt.
- Tướng không được đối mặt.
- Báo chiếu, chiếu hết.
- Ăn quân, lưu biên bản, hoàn tác.
- Đồng hồ 15 phút mỗi bên.
- Lật bàn cờ.

File:
- index.html
- style.css
- script.js
