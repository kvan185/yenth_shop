WEB Ô ĂN QUAN 2 NGƯỜI

Cách chạy:
1. Giải nén ZIP.
2. Mở index.html bằng Chrome, Edge hoặc Firefox.
3. Chọn ô bên mình rồi chọn hướng rải quân.

Tính năng:
- Bàn Ô Ăn Quan truyền thống.
- 10 ô dân + 2 ô quan.
- Rải quân trái/phải.
- Ăn quân và ăn quan.
- Quan tính 10 điểm.
- Tự vay quân khi hết quân bên mình.
- Tính điểm cuối ván.
- Hoàn tác, chơi lại.

Lưu ý:
- Đây là bản web offline viết bằng HTML/CSS/JavaScript.
