# Truth or Dare App

Ứng dụng web mini chơi Truth or Dare bằng HTML, CSS và JavaScript.

## Cách chạy
1. Giải nén file ZIP
2. Mở file `index.html` bằng trình duyệt

## Tính năng
- Chọn chế độ Truth hoặc Dare
- Random câu hỏi/thử thách
- Random cả Truth/Dare
- Copy câu hỏi hiện tại
- Giao diện responsive
