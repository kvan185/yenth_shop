CẦU SINH VÔ TẬN - MINI WEB GAME

Cách chạy:
1. Giải nén file ZIP.
2. Mở file index.html bằng Chrome, Edge hoặc Firefox.
3. Chơi trực tiếp, không cần cài đặt server.

Cấu trúc:
- index.html: giao diện chính
- style.css: hiệu ứng và giao diện kinh dị
- script.js: logic game, tình huống và đáp án

Muốn thêm màn chơi:
Mở file script.js, thêm object mới vào mảng scenarios.
