# Giả Lập Môi Trường Nước

Mini web HTML/CSS/JavaScript thuần.

## Cách chạy

1. Giải nén file ZIP.
2. Mở file `index.html` bằng trình duyệt.
3. Click/kéo chuột trên màn hình để tạo gợn sóng.

## Tính năng

- Nền nước chuyển động.
- Sóng lan khi click hoặc kéo chuột.
- Cá bơi, bọt khí, tia sáng dưới nước.
- Hiệu ứng mưa tạo gợn sóng.
- Nút bật/tắt cá, bật/tắt mưa, đổi màu nước.
