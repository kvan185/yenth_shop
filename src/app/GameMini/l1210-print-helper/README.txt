L1210 Print Helper

Cach dung:
1. Mo file index.html bang Chrome/Edge.
2. Chon file PDF/anh/TXT/HTML can in.
3. Bam Ctrl + P / In file dang xem de mo hop thoai in nhu trinh duyet.
4. Voi PDF, co the tao:
   - PDF 2 trang / 1 mat
   - PDF trang le
   - PDF trang chan
   - PDF trang chan dao nguoc de in 2 mat thu cong

Luu y:
- Trinh duyet khong the dieu khien truc tiep driver Epson L1210.
- Chuc nang xu ly PDF dung thu vien pdf-lib tu CDN, can Internet khi mo lan dau.
- File DOC/DOCX nen xuat sang PDF truoc roi moi in bang app nay.
